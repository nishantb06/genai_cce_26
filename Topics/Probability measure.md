It is a function which maps an element of the Event space to a Positive real number between 0 and 1. The function must satisfy the [[axioms of probability]]

We need to assign a weight / volume to a particular sample from the event or sample space . this function which does this is called the probability measure. 

Now there are certain conditions like the probability measure should be additive. 
If two events are disjoint then the combined probability measure of either of them happening should be the addition of the probability measure of each of them. These set of rules is also called the [[Sigma Algebra]] or the Borel Sigma algebra



### Properties of Probability Measure

$$P : \mathcal{F} \to [0, 1]$$

1. $P(\Omega) = 1,\quad P(\emptyset) = 0$
2. If $A_1 \cap A_2 = \emptyset$, then $P(A_1 \cup A_2) = P(A_1) + P(A_2)$
