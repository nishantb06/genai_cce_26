My understanding:

While [[Sigma Field|σ-fields]] deal with countably infinite sample spaces, the individual elements in the event space are still discrete. For continuous sample spaces — e.g. time on the real line — we use **Borel σ-fields** as the natural event spaces for bounded continuous outcomes.

---

While a σ-field allows us to consider countable collections of events, it is still insufficient for events defined on the real line (e.g. time), because points on $\mathbb{R}$ are not countable. Any non-empty interval is uncountably infinite.

For finite sets and countable sets, the smallest units are the elements themselves — we can count them. On the real line, we cannot. The smallest unit we use to construct a σ-field for the real line is a semi-closed interval $(-\infty, b] = \{x \mid -\infty < x \leq b\}$. The **Borel σ-field** is the σ-field generated by these semi-closed intervals.

### Definition 2.17 (Borel σ-field)

The Borel σ-field $\mathcal{B}$ is the smallest [[Sigma Field|σ-field]] containing all semi-closed intervals $(-\infty, b] = \{x \mid -\infty < x \leq b\}$.

The difference between $\mathcal{B}$ and a general σ-field on a countable $\Omega$ is how we *generate* measurable sets: on countable spaces we build from individual outcomes; on $\mathbb{R}$ we build from semi-closed intervals, then close under complement and countable union/intersection.

Being a σ-field, $\mathcal{B}$ is closed under complement, union, and intersection. In particular, subsets of the following forms are also in $\mathcal{B}$: $(a,b)$, $[a,b]$, $(a,b]$, $[a,b)$, $[a,\infty)$, $(a,\infty)$, $(-\infty,b]$, $\{b\}$. For example, $(a,\infty)$ can be constructed as $(-\infty,a]^c$, and $(a,b]$ as $(-\infty,b] \cap (a,\infty)$.

### Example 2.27

**Waiting for a bus.** Let $\Omega = \{t \mid 0 \leq t \leq 30\}$. The Borel σ-field contains all semi-closed intervals $(a,b]$ where $0 \leq a \leq b \leq 30$. Two possible events:

- $F_1 = \{\text{less than 10 minutes}\} = \{t \mid 0 \leq t < 10\} = \{0\} \cup (\{0 < t \leq 10\} \cap \{10\}^c)$
- $F_2 = \{\text{more than 20 minutes}\} = \{t \mid 20 < t \leq 30\}$

Further discussion of the Borel σ-field can be found in Leon-Garcia (3rd Edition), Chapter 2.9.
